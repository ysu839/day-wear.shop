# DAYWEAR — Digital Flagship

Luxury streetwear digital experience.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for production

```bash
npm run build
npm run preview
```

The `dist/` folder is ready to deploy anywhere.

## Deploy free in 30 seconds

1. Run `npm run build`
2. Go to [netlify.com/drop](https://netlify.com/drop)
3. Drag the `dist/` folder onto the page
4. Done — live at a free `.netlify.app` URL

## Connect payments (Stripe)

1. Sign up at [stripe.com](https://stripe.com)
2. Add Stripe Checkout or Payment Links to the checkout flow
3. Payouts land in your bank automatically
